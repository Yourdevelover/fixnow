Project Kuliah  Website Layanan Teknisi

Proyek ini merupakan tugas kuliah yang dikembangkan menggunakan Laravel sebagai framework utama, dengan MySQL sebagai basis data, serta dihosting pada platform InfinityFree.  
Tujuan utama dari proyek ini adalah untuk mempelajari proses pengembangan aplikasi web modern, mulai dari perancangan, implementasi, hingga deployment ke layanan hosting.

Tujuan
 Mengimplementasikan konsep MVC pada framework Laravel.
 Membangun aplikasi berbasis data dengan MySQL.
 Melakukan deployment aplikasi ke hosting gratis (InfinityFree).
 Menyediakan layanan teknisi online sebagai studi kasus.

Teknologi yang Digunakan
 Framework: Laravel 10
 Database: MySQL
 Hosting: InfinityFree
 Frontend: Bootstrap / Tailwind CSS

Fitur Utama
 Registrasi dan login pengguna
 Pemesanan layanan teknisi secara online
 Manajemen data layanan dan pengguna
 Routing dinamis dengan Laravel
 Deployment ke InfinityFree

Catatan
 Proyek ini dibuat untuk keperluan akademik dan pembelajaran.  
 Tidak ditujukan untuk penggunaan komersial, melainkan sebagai implementasi praktis dari materi perkuliahan.
